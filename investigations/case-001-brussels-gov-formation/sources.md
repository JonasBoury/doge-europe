9. **Protest documentation** - Civil society reactions and demands
10. **Expert commentary** - Political scientists, economists, legal scholars

### Source Verification Standards:

- **Official documents:** Verify authenticity, check for updates
- **Media reports:** Cross-reference across multiple outlets, check primary sources
- **Academic research:** Check peer review status, author credentials
- **Party statements:** Verify official release, check for context manipulation
- **Expert commentary:** Verify credentials, check for conflicts of interest

### Ethical Considerations:

- Protect source identities where confidentiality requested
- Verify leaked documents before use
- Respect privacy regulations (GDPR) for personal data
- Attribute all sources transparently in final report
- Note any paywalled or restricted-access sources
